# FireBase
Fay-yaad Ahmed Parker: ST10434066, PROG7313 ICE04

### This is an Android Kotlin application that uses Firebase Authentication, Firestore, and Firebase Storage to allow users to register, log in, create posts with images, and view posts in a feed.

# Technologies used:
- Kotlin
- Firebase Auth
- Firebase Firestore
- Firebase Storage
- RecyclerView

# Disclaimer
Firebase Storage was not implemented due to regional billing restrictions; therefore, image uploads are not supported in this version of the application.
